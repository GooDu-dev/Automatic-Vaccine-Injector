First Automatic Vaccine Injection Machine Draft

ต้นแบบของเครื่องได้แนวคิดมาจาก เครื่องวัดความดันในกระเเสเลือด โดยได้นำมาประยุกต์ให้เข้ากับปัญหาของการฉีดวัคซีน เช่น ปัญหาทางด้านบุคลากรที่ต้องใช้จำนวนที่เยอะในการฉีดวัคซีน 1 ครั้ง ปัญหาความล่าช้าในการฉีด เนื่องจากบุคลากรทางการเเพทย์ที่มีอยู่จำกัด ทำให้ไม่สามารถฉีดวัคซีนได้เยอะ เเละปัญหาการจัดเก็บข้อมูลของผู้ที่เข้ารับการฉีดวัคซีน โดยเครื่องจะใช้พลังงานไฟฟ้าเป็นหลักทำให้ง่ายต่อการติดตั้งและขนย้าย

Second Vaccine Automatic Injection Machine Draft

จากปัญหาที่กล่าวมาข้างต้นทำให้พวกเราได้คิดเเละวิเคราะห์แนวทางการแก้ไขปัญหาของการฉีดวัคซีน ซึ่งทำให้เราได้โครงร่างของเครื่องฉีดวัคซีนอัตโนมัติในรูปของภาพ Perspective และ Top Front Side Back โดยมีหลักการทำงานดังนี้ ขั้นแรกผู้ที่จะเข้ารับการฉีดวัคซีนต้องไปซื้อขวดบรรจุวัคซีนตามที่แพทย์ได้ชี้แจง เเล้วจึงนำบัตรประชาชนไปเสียบที่เครื่องฉีดวัคซีนอัตโนมัติเเละจึงทำการใส่วัคซีนเข้าไปในเครื่อง(สำหรับเครื่องที่มีการบรรจุวัคซีนเตรียมไว้สำหรับกลุ่มผู้ที่ต้องการจะฉีด เช่น กลุ่มผู้ที่ต้องการจะฉีดวัคซีน COVID-19 ไม่จำเป็นต้องใส่ยาลงในเครื่อง เพราะจะมีขวดวัคซีนเตรียมไว้ในเครื่องเรียบร้อยเเล้ว) หลังจากที่เสียบบัตรประชาชนเข้าไปเเล้ว จะมีข้อมูลปรากฏบนหน้าจอ LCD เพื่อเป็นการยืนยันต่อผู้ที่ต้องการจะฉีดและบุคลการทางการแพทย์ที่ดูแลเครื่องอยู่ โดยข้อมูลนี้จะเชื่อมไปยัง Database ของโรงพยาบาลเพื่อทำการบันทึกข้อมูลว่าผู้ใช้รายนี้ได้ทำการฉีดวัคซีนในวันที่เท่าไร หลังจากนั้นในผู้ใช้ทำการสอดหัวไหล่เข้าไปในเครื่องเเละรอเครื่องทำงาน โดยที่เครื่องจะทำงานโดย ดันแผ่นผ้าไฟเบอร์ให้ชิดกับเเขนโดยมี weight sensor เป็นตัวกำหนดเมื่อได้ขนาดแขนเเล้วหุ่นยนต์จะนำไปคำนวณเพื่อให้ได้จุดที่ควรจะฉีดวัคซีน(บนกล้ามเนื้อ Deltoid) ซึ่งตัวเข็มฉีดยาไร้เข็มจะเลื่อนเข้าหาจุดที่ต้องฉีดยาผ่าน Stepper Motor ที่ต่อเข้ากับสายพานแบบแข็ง เเละตัวขวดวัคซีนจะเคลื่อนที่มาตามรางที่ผู้ใช้ได้ใส่ขวดวัคซีนไปในตอนต้น จนกระทั่งมาถึงตัวเข็มฉีดยาไร้เข็มเเละทำการบรรจุขวดวัคซีนเข้าเข็มฉีดยา เเล้วเข็มจะทำการเคลื่อนที่เข้าหารูที่จะฉีด โดยใช้ Servo Motor เเละรางที่เคื่อนผ่าน Stepper Motor เพื่อที่จะเตรียมตัวเคลื่อนที่ไปหาจุดที่จะฉีด แล้วจึงทำการฉีดขเาไปในกล้ามเนื้อ Middle Deltoid ได้อย่างแม่นยำ

Final Automatic Injection Machine Model

และได้นำแบบโครงร่างที่ได้ออกแบบทั้งคู่นำมาปั้นเป็น Model 3D เพื่อที่จะได้เห็นภาพรวมของเครื่องได้อย่างชัดเจน เช่น ขาที่ยึดเครื่องกับโต๊ะเพื่อไม่ให้เครื่องนั้นขยับ ช่องที่ไว้ใส่วัคซีนเข้าไปในรางวัคซีนที่อยู่ในเครื่อง ช่องเสียบบัตรประชาชนที่เชื่อมต่อเข้ากับ Database ของโรงพยาบาล หน้าจอ LCD ที่ไว้แสดงข้อมูล และปุ่มหยุดการทำงานฉุกเฉินในกรณีที้่เครื่องนั้นเกิดปัญหา หรือเกิดข้อผิดผลาดในการใช้งาน ทำให้เครื่องฉีดวัคซีนอัตโนมัตินี้สามารถช่วยเหลือบุคลากร
ทางการเเพทย์ในการฉีดวัคซีนได้ และได้นำแบบโครงร่างที่ได้ออกแบบทั้งคู่นำมาปั้นเป็น Model 3D เพื่อที่จะได้เห็นภาพรวมของเครื่องได้อย่างชัดเจน เช่น ขา (C clamp) ที่ยึดเครื่องกับโต๊ะเพื่อไม่ให้เครื่องนั้นขยับ ช่องที่ไว้ใส่วัคซีนเข้าไปในรางวัคซีนที่อยู่ในเครื่อง ช่องเสียบบัตรประชาชนที่เชื่อมต่อเข้ากับ Database ของโรงพยาบาล หน้าจอ LCD ที่ไว้แสดงข้อมูล และปุุ่มหยุดทำงานฉุกเฉินในกรณีที่เครื่องนั้นเกิดปัญหา หรือเกิดข้อผิดพลาดในการใช้การ ทำให้เครื่องฉีดวัคซีนอัตโนมัตินี้สามารถช่วยเหลือบุคลากรทางการเเพทย์ในการฉีดวัคซีนได้
